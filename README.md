# Engine

Ce projet est un Moteur graphique utilisant OpenGL 4.2, c++14 && gsl.

# Compilation

## Linux


### Debug

```shell
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=debug ..
make
```

### Release

```shell
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=release ..
make
```
